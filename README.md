# GifCast - animated GIF recorder

## About
A simple tool that allows you to record short animated Gifs from your screen.

## Requirements
### Runtime

  * Gtk 3.?
  * ffmpeg or byzanz?

### Development

 * Vala compiler

## Building
For now just:

    valac --pkg gtk+-3.0 --pkg gmodule-2.0 src/gifcast.vala
