# Peek - animated GIF recorder

## About
A simple tool that allows you to record short animated Gifs from your screen.

## Requirements
### Runtime

  * Gtk 3.10 or higher
  * ffmpeg
  * ImageMagick

### Development

 * Vala compiler
 * CMake >= 2.6

## Building
You can build Peek using CMake:

    cmake . && make
